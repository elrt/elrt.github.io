KISS (Keep It Simple Syntax)

Эзотерический язык программирования.
Вычислительный класс: Тьюринг полный
Расширение файлов: .kiss
Система типов: динамическая,слабая
Система памяти на основе регистров Х/Y
Вдохновлено: Assembly,Brainfuck

Пример кода:

# KISS-compliant "Hello World"
A 72 # 'H'
CP
A 101 # 'e'
CP
A 108 # 'l' (x2)
CP
CP
A 111 # 'o'
CP
A 32 # ' '
CP
A 87 # 'W'
CP
A 111 # 'o'
CP

CP
A 108 # 'l'
CP

CP

JSCN(Just Simple Code Notepad)

Простой редактор кода на джаве:р поддерживает подсветку синтаксиса для таких языков как KISS,Brainfuck,BASIC и Nasm
